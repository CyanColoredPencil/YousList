# YousList

[Adblock Plus][] filter.

[Adblock Plus]: https://adblockplus.org/

## Usage

Install its browser plugin of Adblock Plus and add an subscription as 'YousList' with the url:

```
https://github.com/yous/YousList/raw/master/youslist.txt
```
